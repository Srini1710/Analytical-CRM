# Analytical-CRM
Analysis of Employee Attrition
Files
CRM.py - reads data from Attrition.csv file, contains code that perform analysis using decision tree classifier.
pearson.py - reads data from Attrition.csv file, contains code for pearson correlation coefficient between variables in the file.
Randomforest.py - reads data from Attrition.csv file, contains code that perform analysis using random forest classifier.
Attrition.csv - file containing data related to employee attrition.
